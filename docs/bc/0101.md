Machine Learning (ML) 과 Causal Learning 의 차이점은

- Causal 모델은 기본적으로 확률론적(probabilistic) 모델보다 더 많은 정보를 담게 된다. 왜냐하면 Causal 모델은 조정(intervention)에 의한 효과 또는 분포의 변화를 분석할 수 있어야 하기 때문이다.
